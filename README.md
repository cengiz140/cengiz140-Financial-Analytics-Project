# 📊 Financial Performance Dashboard (Power BI)

This project presents an interactive Power BI dashboard designed to analyse business performance across sales, profit, and operational efficiency. The dashboard provides insights into revenue trends, profitability drivers, and the impact of discounting strategies.

---

## 📌 Project Overview

The dashboard was built using a financial dataset to explore how different factors such as segments, products, regions, and discounts influence overall business performance.

It focuses on answering key business questions:
- Where is revenue being generated?
- Which areas are most profitable?
- How do discounts affect profitability?
- How has performance changed over time?

## 🛠 Tools & Skills Used

- **Power BI**
  - Data modelling (fact table + date table)
  - DAX measures (YTD, YoY %, Profit Margin)
  - Interactive dashboard design

- **Power Query**
  - Data cleaning and transformation
  - Handling formatting issues and datatype conversion

- **Data Analysis**
  - Trend analysis
  - Profitability analysis
  - Business insight generation

---

## 📊 Key Insights

- Strong revenue performance does not always translate into high profitability  
- Higher discount levels increase sales but reduce profit margins  
- Certain segments and products outperform others in terms of efficiency  
- Consistent growth trends observed over time
